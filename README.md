## 👨‍💻 À propos

Développeur **Rust** et **Python**, passionné par le bas niveau, les systèmes embarqués et la cybersécurité offensive.

Je conçois des applications performantes — du firmware no_std pour microcontrôleurs aux moteurs 3D avec rendu GPU, en passant par le backend web et les applications desktop.

- 🌱 **En apprentissage :** CTF, pentest avancé, reverse engineering

- 📝 **Blog technique :** [theora59-dev.github.io/blog](https://theora59-dev.github.io/blog/)
